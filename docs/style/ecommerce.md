# E-commerce Style

## Overview

E-commerce websites are designed to showcase products and drive conversions. They combine attractive product presentation with intuitive navigation and seamless purchasing experiences.

## Key Characteristics

- **Product-focused imagery** with multiple views
- **Clear pricing** and availability information
- **Prominent shopping cart** and checkout flow
- **Category navigation** with filters and sorting
- **Trust signals** (reviews, secure payment, returns)
- **Mobile-optimized** for on-the-go shopping

## Prompts

### Modern Fashion Store
```
Create an e-commerce website for a fashion brand. Include
a hero with featured collection, product grid with quick-view
functionality, size guides, customer reviews, and a sticky
add-to-cart button. Use high-fashion photography with a
minimal black and white design. Include wishlist and
recently viewed features.
```

### Electronics Store
```
Design an electronics e-commerce site with a tech-forward
aesthetic. Include comparison features, detailed spec tables,
video product demos, Q&A sections, and prominent ratings.
Use a dark theme with blue accents. Add a mega-menu for
categories and a powerful search with filters.
```

### Artisan Marketplace
```
Build an artisan marketplace website with a warm, handcrafted
feel. Include seller profiles, product stories, material
details, and a "shop by maker" feature. Use earthy tones
and textured backgrounds. Emphasize uniqueness and
craftsmanship in the design.
```

## Essential Elements

- Product image galleries with zoom
- Size/variant selectors
- Add to cart with quantity
- Customer reviews and ratings
- Related products section
- Secure checkout indicators
- Shipping information
- Return policy links
