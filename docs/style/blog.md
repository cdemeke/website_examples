# Blog Style

## Overview

Blog and magazine websites prioritize readability and content discovery. They use clean typography, organized layouts, and intuitive navigation to keep readers engaged and exploring.

## Key Characteristics

- **Typography excellence** with readable fonts and sizing
- **Content hierarchy** with featured and recent posts
- **Category organization** for content discovery
- **Author attribution** and profile sections
- **Social sharing** integration
- **Comment systems** for community engagement

## Prompts

### Personal Blog
```
Create a personal blog with a warm, inviting design.
Use a single-column layout for posts with a readable
serif font for body text. Include a sidebar with
about section, recent posts, and categories. Add a
newsletter signup form and social links. Use a
light cream background with dark text.
```

### Tech Publication
```
Design a tech news publication website with a modern,
editorial look. Include a featured article hero,
category tabs, article cards with thumbnails,
trending sidebar, and author pages. Use a clean
sans-serif font with a white background and subtle
gray accents. Add reading time estimates and
code snippet styling.
```

### Lifestyle Magazine
```
Build a lifestyle magazine website with a visually-rich,
Pinterest-style layout. Use a masonry grid for articles,
large featured images, category color coding, and
celebrity-style author bylines. Include trending topics,
newsletter popup, and social media feed integration.
```

## Color Palettes

### Classic Editorial
- Background: `#FFFFFF`
- Text: `#1A1A1A`
- Accent: `#C41E3A`
- Secondary: `#666666`

### Warm Personal
- Background: `#FDF6E3`
- Text: `#3D3D3D`
- Accent: `#D35400`
- Link: `#2980B9`

### Modern Publication
- Background: `#F8F9FA`
- Text: `#212529`
- Accent: `#0066CC`
- Secondary: `#6C757D`

## Typography Recommendations

### Body Text
- Size: 18-20px
- Line height: 1.6-1.8
- Max width: 680-720px

### Headlines
- Clear hierarchy (H1 > H2 > H3)
- Sufficient spacing between sections
- Consider contrast between heading and body fonts

## Best For

- Personal bloggers
- Online magazines
- News publications
- Company blogs
- Recipe sites
- Travel journals
- Tutorial sites
